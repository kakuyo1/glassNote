# glassNote 可选后续功能（有趣版）

本文档基于当前版本，扩展一批更有趣、可形成产品差异化的功能方向。

## 当前基线（截至目前）

- 基于 Qt Widgets + C++ 的桌面便签应用
- 透明无边框悬浮窗口，支持窗口位置/尺寸持久化
- 多事项纵向看板，支持新建/删除/清空空事项
- 每条事项独立 Hue 配色，支持全局 UI 缩放
- 本地 JSON 自动保存与恢复
- 事项编辑模式带富文本工具栏
- 基础入场/删除/重排/悬停动画
- 已补齐质量基线：单元测试 / 集成测试 / 性能测试拆分、load-edit-save-reload 流程验证、核心加载保存日志策略

## 有趣功能探索池

### 1）极速捕获（让“记一条”比打开聊天框还快）

- 全局快捷键呼出“极速输入条”（输入回车直接落到看板）
- 剪贴板收件箱（检测新复制文本，一键转便签）
- 拖拽文本/图片到窗口边缘自动创建新事项
- 可选截图 OCR 入箱（实验项，默认关闭）

预期价值：
- 显著提高记录频率，减少思路中断

### 2）专注节奏（从便签到行动）

- 事项绑定番茄计时（25/5 可配），完成自动打点
- 今日专注时间线（按事项聚合）
- 每日回顾卡片（完成数、新增数、最长专注项）
- 到点轻提醒（不抢焦点，仅轻量浮层）

预期价值：
- 把“写下任务”转化为“持续推进任务”

### 3）空间化组织（比单列更有结构）

- 多泳道看板（Today / Next / Waiting / Someday）
- 事项分组与折叠（项目视角）
- 事项关联（A 依赖 B、引用跳转）
- 时间轴回放（查看某一天看板状态）

预期价值：
- 提升复杂任务管理能力，同时保留便签的轻量感

### 4）聪明但可控的智能辅助（可离线优先）

- 从长文本自动提取行动项（动词 + 截止语义）
- 重复事项检测与合并建议
- 一键改写语气（简洁版 / 行动导向版 / 汇报版）
- 周报草稿生成（按完成事项自动拼接）

预期价值：
- 降低整理成本，提高“写完即能用”的效率

### 5）表达力与可玩性（记得住，也愿意用）

- 最近使用组合（自动记录风格/色相/贴纸）
- 轻量贴纸/符号标记（不引入复杂附件系统）
- 成就反馈（连续完成天数、周目标达成徽章）

预期价值：
- 提高使用粘性，让工具更有“个人空间”属性

### 6）外部互通（融入已有工作流）

- 导出为 Markdown 包（含元数据与顺序）
- 深链支持（`glassnote://note/{id}`）
- 拖拽生成可分享快照图（PNG）
- Webhook/命令行入口（外部系统可投递便签）

预期价值：
- 降低切换成本，便于与其他工具协作

### 7）可靠性进阶（“再也不怕丢”）

- 追加写日志（append-only journal）+ 定期快照
- 崩溃恢复模式（启动后可回看最近 N 次自动保存）
- 只读恢复视图（数据异常时先保住可读性）
- 便携模式（单目录携带配置与数据）

预期价值：
- 提升信任感，适合长期积累型使用者

### 8）多人/跨端（远期探索）

- 局域网临时协作板（同网段共享）
- WebDAV/Git 目录同步（可选高级模式）
- 冲突可视化（按卡片维度选择保留版本）
- 只读 Web 预览页（分享进度，不暴露编辑能力）

预期价值：
- 从个人工具扩展到小团队协作场景

## 建议优先路线（趣味 x 成本 x 风险）

若只做 1~2 个迭代，建议优先：

1. 极速输入条 + 全局快捷键
2. 多泳道看板（Today/Next）
3. 每日回顾卡片
4. 事项绑定番茄计时
5. Markdown 导出 + 深链

## 轻量里程碑建议

### 里程碑 A（1~2 周：快速见效）

- 极速输入条
- 全局快捷键新建
- 每日回顾卡片（基础统计）

### 里程碑 B（2~3 周：结构升级）

- 多泳道看板
- 分组折叠
- 拖拽交互细化

### 里程碑 C（2~4 周：差异化体验）

- 番茄计时绑定事项
- 自动行动项提取（可选实验）
- Markdown 导出 + 深链

### 里程碑 D（远期：平台能力）

- 追加写日志 + 恢复模式
- 同步方案探索（WebDAV/Git）
- 只读分享页

## 可直接建 Issue 的 8 个候选

1. 全局快捷键呼出极速输入条
2. Today/Next 双泳道切换
3. 每日回顾弹层（完成/新增/专注）
4. 事项番茄计时与完成打点
5. Markdown 导出（含顺序与样式元数据）
6. 深链跳转到指定事项
7. 追加写日志与崩溃恢复入口
8. 重复事项检测与合并建议
