# glassNote 可选后续功能

本文档基于项目当前状态，整理可选的后续开发项。

## 当前基线（截至目前）

- 基于 Qt Widgets + C++ 的桌面便签应用
- 透明无边框悬浮窗口，支持窗口位置/尺寸持久化
- 多事项纵向看板，支持新建/删除/清空空事项
- 每条事项独立 Hue 配色，支持全局 UI 缩放
- 底层透明度百分比调节菜单
- 本地 JSON 自动保存与恢复
- 事项编辑模式带富文本工具栏（加粗、斜体、下划线、删除线、项目符号、数字编号）
- 基础入场/删除/重排/悬停动画

## 可选功能待办

### 1）编辑体验

- 增加格式快捷键（`Ctrl+B`、`Ctrl+I`、`Ctrl+U`、`Ctrl+Shift+X`）
- 增加清除格式与段落对齐工具
- 可选清单模式（可勾选复选框列表）
- 优化列表编辑行为（`Enter` / `Backspace` 一致性）
- 可选事项内搜索与高亮

预期价值：
- 提升日常编辑效率，降低输入摩擦

### 2）数据与可靠性

- 支持 JSON 导出/导入（合并或覆盖模式）
- 本地备份快照与一键恢复
- JSON 损坏时的回退与恢复提示
- 可选文件变更监听，支持外部修改同步

预期价值：
- 提升数据安全、迁移能力与恢复信心

### 3）效率功能

- 全局快捷键快速新建事项
- 系统托盘集成（显示/隐藏、快速新建、退出）
- 置顶开关与窗口锁定模式
- 可选事项提醒时间

预期价值：
- 更好融入日常工作流，提升记录速度

### 4）视觉与交互打磨

- 在 Windows 上提供更真实的亚克力/模糊效果路径
- 主题预设与可选自定义取色
- 随 UI 缩放优化字体与间距预设
- 细化动画时序与动效一致性

预期价值：
- 提升产品识别度与整体质感

### 5）架构与质量

- 为序列化、事项排序、控制器逻辑增加单元测试
- 增加 load/edit/save/reload 流程集成测试脚本
- 制定启动/加载/保存错误日志策略
- 增加大量事项场景的性能检查

预期价值：
- 降低回归风险，提升长期可维护性

## 建议优先级

若只选少量功能，建议按如下顺序推进（收益/成本比更高）：

1. 格式快捷键
2. 导出/导入 + 备份快照
3. 列表编辑行为优化
4. 托盘 + 快速新建快捷键
5. 序列化/控制器单元测试

## 轻量里程碑建议

### 里程碑 A（编辑质量）

- 快捷键 + 清除格式
- 列表行为优化
- 工具栏交互细节打磨

### 里程碑 B（数据安全）

- 导出/导入
- 备份快照
- JSON 异常恢复回退

### 里程碑 C（日常工作流）

- 托盘集成
- 快速新建快捷键
- 置顶 / 锁定模式
